# Personal Website

Welcome to my personal website repository! This project aims to showcase my portfolio and personal projects.

## Project Structure

- `/assets` - Contains images, CSS, and JavaScript files.
- `/pages` - HTML files for different pages of the website.
- `index.html` - The main landing page.

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/1206Jars/Personal-Website.git
   ```
2. Open `index.html` in your browser to view the website.
